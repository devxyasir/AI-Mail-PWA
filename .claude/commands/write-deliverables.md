# write-deliverables

## Purpose

Write or update all final assignment deliverable documents.

## Agent to Use

documentation

## Steps

1. Read the full codebase to understand what was actually built.
2. Update docs/architecture.md:
   - Frontend architecture
   - Backend architecture
   - Provider adapter pattern
   - AI pipeline
   - Database schema
   - Deployment
3. Update docs/agents.md:
   - All 8 agents with descriptions
   - All 6 skills
   - All 6 commands
   - All 3 hooks
4. Update docs/workflow.md:
   - Claude Code CLI workflow used
   - Model selection per phase
   - Specs-driven development approach
   - How agents were used
5. Update docs/testing.md:
   - Test strategy
   - Test results
   - Coverage notes
6. Update README.md:
   - Project description
   - Setup instructions
   - Live Vercel URL
   - Key features
7. Verify CLAUDE.md is current.

## Output Format

- docs/architecture.md (complete)
- docs/agents.md (complete)
- docs/workflow.md (complete)
- docs/testing.md (complete)
- README.md (updated)

## Constraints

- Only document what was actually implemented.
- Be honest about partial implementations.
- Do not invent features.

## Usage Example
/write-deliverables final assignment submission documents
